## surround\_modes

This is a child capability of `has_discrete_surround_mode_select`. It is the parent XML entry for all of the surround modes (`<surround_mode>`)defined by the driver. 

### Signature

`<surround_modes></surround_modes>`


| Parameter | Description           |
| --------- | --------------------- |
| XML       | Surround Mode entries |


### Example

```xml
<capabilities>
   <has_discrete_surround_mode_select>True</has_discrete_surround_mode_select>
		<surround_modes>
			<surround_mode></surround_mode>
	</surround_modes>
</capabilities>
```
