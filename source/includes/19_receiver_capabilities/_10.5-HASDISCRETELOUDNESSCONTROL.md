## has\_discrete\_surround\_mode\_select

This capability must be declared True to support surround modes in conjunction with the use of the receiver proxy. 

### Signature

`<has_discrete_surround_mode_select></has_discrete_surround_mode_select>`


| Parameter | Description |
| --------- | ----------- |
| bool      | True/False  |


### Example

```xml
<capabilities>
   <has_discrete_surround_mode_select>True</has_discrete_surround_mode_select>
</capabilities>
```
